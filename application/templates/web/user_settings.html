Wygeneruj nowy token lub użyj istniejącego.

1. Czym jest token i czy mogę go zmienić?

Token to identyfikator, który umożliwia grupowanie wpisów i komentarzy użytkownika zachowując jego anonimowość - Token widoczny jest tylko dla Ciebie.

2. Co stanie się gdy zmienię token?

Po wygenerowaniu tokenu będziesz postrzegany przez stronę jako nowy użytkownik.

3. Jak działa token?

Każda przeglądarka internetowa generuje unikalny token i zapisuje go w ciasteczkach. Jeśli chciałbyś napisać komentarz z innej przeglądarki niż obecnie używana, oraz chciałbyś pozostać przy tym identyfikowany jako ten sam użytkownik, musisz skopiować ten token i wpisać go w innej przeglądarce/urządzeniu w tym widoku po wciśnięciu "użyj istniejącego".
