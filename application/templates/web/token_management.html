Czym jest token i czy mogę go zmienić?

Token jest identyfikatorem, dzięki któremu portal grupuje wpisy i komentarze jednego użytkownika pozwalając mu zachować anonimowość.

Token widoczny jest tylko dla ciebie w widoku dodawania nowego wpisu lub komentarza.

Token pozwala na oznaczenie komentarzy autora wpisu pod jego wpisem, a także umożliwia oznaczenie na stronie twoich wpisów i komentarzy (oznaczenie widoczne jest tylko dla ciebie).

Każda przeglądarka internetowa generuje unikalny token i zapisuje go w ciasteczkach. Jeśli chciałbyś napisać komentarz z innej przeglądarki niż obecnie używana, oraz chciałbyś pozostać przy tym identyfikowany jako ta sama osoba, musisz skopiować ten token i wpisać go przy dodawaniu wpisu lub komentarza.

Zmiana tokena spowoduje, że będziesz postrzegany przez stronę jako nowy inny użytkownik.
